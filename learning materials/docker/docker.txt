docker下载 redis镜像： docker pull redis:3.2
docker 挂载安装redis：
docker run -d --privileged=true -p 6379:6379 -v /docker/redis/conf/redis.conf:/etc/redis/redis.conf -v /docker/redis/data:/data --name redis01 redis:3.2 redis-server /etc/redis/redis.conf 
docker 启动 redis ：  docker exec -it 7d6606fe1582 redis-cli

docker下载 mysql镜像： docker pull mysql:5.6.22
docker 挂载安装mysql： docker run -p 3306:3306 --name mysql -v $PWD/conf:/etc/mysql/conf.d -v $PWD/logs:/logs -v $PWD/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=root -d mysql:5.6
(注意PWD为当前路径)

docker下载 rabbitMQ镜像： docker pull rabbitmq:3.7
docker 安装rabbitMQ：
	docker run -d -p 5672:5672 -p 15672:15672 --name rabbitmq rabbitmq:3.7
